# Iteration Layer Skills

Skills for [Claude Code](https://claude.ai/code) and AI agents to integrate [Iteration Layer](https://iterationlayer.com) — composable APIs for document, website, image, and sheet processing.

## Installation

```
/plugin marketplace add iterationlayer/skills
```

Then install the plugins you need:

```
/plugin install iterationlayer@iterationlayer-skills
/plugin install document-extraction@iterationlayer-skills
/plugin install website-extraction@iterationlayer-skills
/plugin install image-editing@iterationlayer-skills
/plugin install image-design@iterationlayer-skills
/plugin install document-creation@iterationlayer-skills
/plugin install spreadsheet-generation@iterationlayer-skills
```
